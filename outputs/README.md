# DeskVault Support Portal CTF

Beginner web CTF challenge for the assignment **Design & Develop a CTF Challenge**.

## Concept

The challenge teaches why hiding a signup page in front-end HTML is not real access control.

## How to Run

No backend or install step is required.

1. Open `challenge/index.html` in a web browser.
2. Try signing in with any username and password.
3. Use browser DevTools to find and reveal the hidden signup portal.
4. Create an account in the portal to reach the verification page and flag.

The flag format is `FLAG{some_text_here}`.

## Files

- `challenge/index.html` - web page structure
- `challenge/style.css` - simple themed styling
- `challenge/app.js` - challenge logic
- `SOLUTION.md` - step-by-step solution
- `DESIGN.md` - concept and learning goals
